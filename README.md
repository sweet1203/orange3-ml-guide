# 🍊 Orange3 기계학습 실습 가이드

고등학교 「인공지능 기초」 4-2 기계학습 단원용 Orange3 실습 가이드 모음입니다.
코딩 없이 위젯을 연결해 **분류 · 비지도(군집) · 회귀**를 직접 체험합니다.

## 🔗 바로 보기 (GitHub Pages)

👉 **https://sweet1203.github.io/orange3-ml-guide/**

## 📚 구성

| 가이드 | 유형 | 데이터 |
|---|---|---|
| 동물 분류 AI | 분류(지도) | zoo |
| 붓꽃 끼리끼리 묶기 ⭐ | 비지도(군집) | iris |
| 끼리끼리 묶기(맛보기) | 비지도(군집) | mathematics-football |
| 광고비로 판매량 예측 | 회귀(지도) | advertising |
| **SDGs × 기계학습 종합 과제** 🌍 | 분류·회귀·군집(선택) | 내장 Datasets(직접 선택) |

> 종합 과제(`과제_SDGs기계학습_내장데이터셋.html`)는 교과서 4-2-4단원(기계학습 활용·SDGs)을 따라, 학생이 SDG 목표를 하나 골라 어울리는 내장 데이터셋으로 직접 AI를 만드는 활동입니다. **SDG ↔ 내장 데이터셋 참고표**가 포함되어 있습니다.

- 모든 데이터는 Orange3 **Datasets 위젯**에 내장/제공되어 별도 다운로드가 필요 없습니다.
- 위젯 입출력(연결 방향)은 [biolab/orange3](https://github.com/biolab/orange3) 소스 기준으로 표기했습니다.

## ℹ️ 사용

`index.html`을 열면 목차에서 각 실습으로 이동할 수 있습니다.
인터넷 브라우저만 있으면 되며, 외부 라이브러리 의존성이 없습니다.
