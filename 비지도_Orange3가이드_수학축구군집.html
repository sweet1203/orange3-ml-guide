🎯 오늘의 목표 — 학생들의 수학 실력·축구 실력만 보고, 정답(그룹)을 안 알려줘도 컴퓨터가 비슷한 학생끼리 무리(군집)로 묶는 걸 직접 확인한다.

1 어떤 데이터를 쓰나요? ⚽📐

mathematics-football — Orange가 "교실에서 군집을 라이브로 시연하라고" 만든 작은 연습 데이터예요. 학생 11명, 각자 두 가지 점수만 있어요.

학생 이름 (메타) 📐 mathematics (수학 실력) ⚽ football (축구 실력)

정답(어느 그룹인지)은 없습니다. → 그래서 비지도학습(군집)이에요.

🟠 중요 — File 위젯 말고 Datasets 위젯!
Orange가 제공하는 예제 데이터는 Datasets 위젯에 들어 있어요.
· File = 내 컴퓨터에 있는 파일(csv 등)을 여는 위젯
· Datasets = Orange 서버가 제공하는 큐레이션 예제 목록(설명 포함)을 내려받는 위젯 ← 오늘 사용

2 Datasets 위젯으로 데이터 불러오기 📥

  1. 왼쪽 위젯 목록 Data 칸에서 Datasets 위젯을 캔버스에 놓아요. (File 아님!)
  2. Datasets를 더블클릭 → 인터넷에서 받아온 예제 데이터 목록이 표로 떠요. (각 줄에 설명·출처가 있음)
  3. 위쪽 검색창에 mathematics 라고 입력 → mathematics-football 줄을 찾기.
  4. 그 줄을 더블클릭(또는 선택 후 창 닫기) → 처음이면 자동으로 다운로드(구름→체크 표시) 돼요.
  5. 이제 Datasets 위젯이 그 데이터를 Data로 내보낼 준비 완료!
🌐 Datasets 위젯은 인터넷이 필요해요(처음 받을 때). 한 번 받으면 캐시에 저장돼 다음엔 빨라요.

3 위젯 연결, 이렇게 해요 🔗

이렇게 하면이런 일이
위젯 오른쪽 가장자리의 반원(점)에서 시작해 다른 위젯으로 드래그두 위젯이 선으로 연결돼요 (데이터가 흐름)
연결선을 더블클릭어느 출력→입력을 이을지 고르는 창이 열려요
선을 클릭하고 Delete연결이 끊겨요
💡 연결은 항상 “출력(오른쪽) → 입력(왼쪽)” 방향이에요. 아래 단계마다 어느 점에서 어느 점으로 잇는지 정확히 표시해 둘게요.

5 단계별 따라하기 👣

1

데이터 표로 확인

Data Table · Data
DatasetsData ▸──▶▸ DataData Table
  1. Data Table을 놓고 위 방향대로 연결 → 더블클릭.
  2. 학생 11명의 수학·축구 점수가 표로 보이면 성공. (정답 열은 없어요!)
2

묶기 전 점 보기

Scatter Plot · Visualize
DatasetsData ▸──▶▸ DataScatter Plot
  1. Scatter Plot을 놓고 Datasets와 연결 → 더블클릭.
  2. 축 설정 → Axis x: mathematics, Axis y: football.
  3. 점 11개가 흩어져 보여요. (아직 색 구분 없음)
3

군집 만들기

k-Means · Unsupervised
DatasetsData ▸──▶▸ Datak-Means
  1. k-Means를 놓고 Datasets와 연결. (왼쪽 Unsupervised 칸)
  2. 더블클릭 → Number of Clusters: 3 (Fixed에 3 입력).
  3. 이제 데이터에 ‘Cluster(군집)’ 정보가 붙어 함께 출력돼요.

k-평균: 중심점 k개를 찍고 → 가까운 점끼리 모으고 → 중심을 옮기길 반복.

4

묶은 결과 색으로 보기

Scatter Plot (1) · Visualize
k-MeansData ▸──▶▸ DataScatter Plot (1)⬅ 이번엔 k-Means에서 연결! (Datasets 아님)
  1. 새 Scatter Plot을 놓고 k-Means와 연결.
  2. 더블클릭 → x: mathematics, y: football, Color: Cluster.
  3. 🎉 점들이 세 가지 색으로 묶였어요! (예: 공부파 / 운동파 / 둘 다 잘함)

6 도전 & 확인하기 ✅

  • 군집 개수를 2 또는 4로 바꾸면 색이 어떻게 변하나요?
  • 각 색 그룹은 어떤 학생들인가요? (예: 수학↑축구↓)
  • 내 점수(수학·축구)를 상상해 표에 넣으면 어느 그룹일까?
  • 비지도학습엔 왜 ‘정확도’를 바로 못 구할까? (정답이 없으니까!)

7 실습 결론 질문 🧐 (실습 끝나고 답해보기)

Q1. 정답(어느 그룹인지)을 한 번도 안 알려줬는데도 학생들이 무리로 묶였나요? 이것이 비지도학습에 대해 말해주는 바는 무엇일까요?

Q2. 각 색 군집은 어떤 특징의 학생 무리였나요? (수학·축구 점수로 설명해보기)

Q4. 비지도학습은 왜 ‘정확도(%)’를 바로 구하기 어려울까요? 분류(지도학습)와의 결정적 차이는?
