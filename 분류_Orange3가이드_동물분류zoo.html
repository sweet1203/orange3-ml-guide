🎯 오늘의 목표 — 동물의 특징(젖을 먹이나? 깃털이 있나?…)을 보고 동물 종류(포유류·새·물고기…)를 맞히는 AI를 만들고, 그 AI의 정확도와 판단 규칙을 직접 확인한다.

1 zoo 데이터가 뭐예요? 🐾

동물 101마리가 한 줄씩 들어 있고, 각 동물마다 16가지 특징과 맨 끝에 정답(종류)이 붙어 있어요.

📋 16가지 특징 (대부분 예/아니오)

털 hair깃털 feathers알 eggs젖 milk 날기 airborne물속 aquatic사냥 predator이빨 toothed 등뼈 backbone숨쉬기 breathes독 venomous지느러미 fins 다리 수 legs꼬리 tail애완 domestic고양이크기 catsize

✅ 정답 라벨 = type (7종)

포유류 mammal새 bird물고기 fish양서류 amphibian 파충류 reptile곤충 insect무척추 invertebrate
🟠 중요 — File 위젯 말고 Datasets 위젯!
· File = 내 컴퓨터에 있는 파일(csv 등)을 여는 위젯
· Datasets = Orange가 제공하는 예제 데이터 목록(설명 포함)을 내려받는 위젯 ← 오늘 사용
👉 정답(type)이 있으니까 = 지도학습(분류)! 정답 없이 묶기만 했던 군집(비지도)과 정반대예요.

2 Datasets 위젯으로 불러오기 📥

  1. 왼쪽 Data 칸에서 Datasets 위젯을 캔버스에 놓아요. (File 아님!)
  2. 더블클릭 → 예제 데이터 목록이 표로 떠요(인터넷 필요) → 검색창에 zoo 입력.
  3. zoo 줄을 더블클릭 → 처음이면 자동 다운로드(구름→체크).
  4. 아래에 101 instances, target: type 이 보이면 성공 🎉 이제 Data로 내보낼 준비 완료.
🌐 처음 받을 때만 인터넷이 필요하고, 이후엔 캐시를 써요. (서버: datasets.biolab.si)

  • 📊 Visualize — 그래프

연결선을 더블클릭어느 출력→입력 채널을 이을지 고르는 창

💡 아래 단계마다 어느 출력 점에서 어느 입력 점으로 잇는지 칩으로 정확히 표시했어요. 색이 다른 칩(Learner·Tree·Evaluation Results)은 Data가 아닌 특별한 채널이에요.

4 완성할 모양 & 연결 정리 🗺️

📌 연결 8개 — 정확한 방향 (출력 ▸ ──▶ ▸ 입력)

DatasetsData ▸──▶▸ DataData Table
DatasetsData ▸──▶▸ DataTree
TreeModel ▸──▶▸ TreeTree Viewer⬅ Tree의 Model 출력이 Tree Viewer의 Tree 입력으로 연결돼요 (자동)
DatasetsData ▸──▶▸ DataTest and Score
TreeLearner ▸──▶▸ LearnerTest and Score⬅ 핵심! 학습기는 Data가 아니라 Learner 선으로 연결돼요.
kNNLearner ▸──▶▸ LearnerTest and Score
Naive BayesLearner ▸──▶▸ LearnerTest and Score
Test and ScoreEvaluation Results ▸──▶▸ Evaluation ResultsConfusion Matrix
🟠 연결이 엉뚱한 채널로 붙으면 연결선을 더블클릭해서 위 표대로 맞추세요. (특히 학습기→Test and Score는 Learner→Learner!)

5 단계별 따라하기 👣

1

데이터 표로 확인

Data Table · Data
DatasetsData ▸──▶▸ DataData Table
  1. Data Table을 놓고 연결 → 더블클릭.
  2. 16가지 특징과 맨 끝 type(정답)을 눈으로 확인.

예: milk=1, hair=1 인 동물은 type이 mammal(포유류)이죠?

2

분류 AI 학습

Tree · Model
DatasetsData ▸──▶▸ DataTree
  1. Tree 위젯을 놓고 Datasets와 연결. (왼쪽 Model 칸)
  2. 특징을 보고 “이러면 포유류, 저러면 새” 규칙을 스스로 배워요.
3

판단 규칙 눈으로 보기

Tree Viewer · Visualize
TreeModel ▸──▶▸ TreeTree Viewer
  1. Tree Viewer를 놓고 Tree와 연결 → 더블클릭.
  2. 나뭇가지처럼 펼쳐진 판단 규칙이 보여요.

🌟 예: “젖을 먹이나요(milk)? → 예 → 포유류”. AI가 어떻게 결정하는지 사람도 이해할 수 있어요!

4

정확도 재기

Test and Score · Evaluate
DatasetsData ▸──▶▸ DataTest and Score
TreeLearner ▸──▶▸ LearnerTest and Score
  1. Test and Score를 놓고 두 줄을 위처럼 연결. (Evaluate 칸)
  2. 더블클릭 → Cross validation 선택 → CA(정확도) 확인. (보통 90%↑)

Cross validation = 데이터를 나눠 번갈아 시험 보는 공정한 채점법.

5

모델 비교하기 🆚

kNN · Naive Bayes · Model
kNNLearner ▸──▶▸ LearnerTest and Score
Naive BayesLearner ▸──▶▸ LearnerTest and Score
  1. kNN·Naive Bayes를 추가로 놓고 둘 다 Test and Score에 Learner로 연결.
  2. 표에 세 모델의 CA가 나란히 → 누가 제일 잘 맞히나 비교!
6

헷갈린 동물 찾기

Confusion Matrix · Evaluate
Test and ScoreEvaluation Results ▸──▶▸ Evaluation ResultsConfusion Matrix
  1. Confusion Matrix를 놓고 Test and Score와 연결 → 더블클릭.
  2. 어떤 종류를 헷갈렸는지 표로 확인. (대각선=정답, 나머지=실수)

7 자주 나오는 문제 (교사용) 🛟

🟠 펼쳐서 보기 — 잘 안 될 때 점검
Datasets 목록이 안 떠요
인터넷 연결 확인(처음 받을 때 온라인 필요). 검색창에 zoo.
Tree / kNN 위젯이 안 보여요
왼쪽 Model 칸 확인.
Test and Score에 점수가 안 나와요
Datasets→Data와 Tree→Learner가 둘 다 연결됐는지 확인.
Tree Viewer가 비어 있어요
Tree의 Model이 Tree Viewer의 Tree 입력에 연결됐는지(연결선 더블클릭으로 확인), Tree에 Datasets가 연결돼 학습이 됐는지 확인.
Confusion Matrix가 비어 있어요
Test and Score(Datasets 아님!)의 Evaluation Results에 연결했는지 확인.

상단 Windows → Display Widgets on Top.
