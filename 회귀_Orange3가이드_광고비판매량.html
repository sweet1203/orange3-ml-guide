광고비를 넣으면 판매량을 예측하는 AI를 만든다 — 회귀(지도학습) 실습 · 선형회귀

🍊 도구: Orange3 📦 데이터: advertising 🧩 Datasets 위젯으로 불러옴 ⏱️ 약 50분 ⌨️ 코딩 0줄
🏠 실습 목록으로
🎯 오늘의 목표 — TV·라디오·신문 광고비로 판매량(Sales)을 예측하는 선형회귀 AI를 만들고, 그 예측이 얼마나 정확한지(R²)까지 확인한다.

1 어떤 데이터를 쓰나요? 📺📻📰

advertising — 가게 200곳의 광고비와 그 결과 판매량이 적혀 있어요. "광고를 많이 하면 더 팔릴까?"를 알아보는 데이터예요.

입력(특징) 3가지 — 광고비

📺 TV📻 Radio📰 Newspaper

예측 대상(정답) — 숫자!

💰 Sales (판매량)
👉 맞히려는 정답이 숫자(연속값)예요. 종류를 맞히면 분류, 숫자를 맞히면 회귀! (지도학습은 둘 다 정답이 있어요.)
🟢 중요 — File 위젯 말고 Datasets 위젯!
· File = 내 컴퓨터 파일을 여는 위젯
· Datasets = Orange가 제공하는 예제 데이터 목록을 내려받는 위젯 ← 오늘 사용

2 Datasets 위젯으로 불러오기 📥

  1. 왼쪽 Data 칸에서 Datasets 위젯을 캔버스에 놓아요. (File 아님!)
  2. 더블클릭 → 예제 목록 표가 떠요 → 검색창에 advertising 입력.
  3. advertising 줄을 더블클릭 → 처음이면 자동 다운로드(구름→체크).
  4. 이제 Datasets가 데이터를 Data로 내보낼 준비 완료!
🌐 처음 받을 때 인터넷 필요(서버: datasets.biolab.si). 한 번 받으면 캐시 사용.

DatasetsData ▸──▶▸ DataScatter Plot

DatasetsData ▸──▶▸ DataTest and Score⬅ 데이터(훈련용)를 평가기로
Linear RegressionLearner ▸──▶▸ LearnerTest and Score⬅ 핵심! 학습기는 Data가 아니라 Learner 선으로 연결돼요. (Test and Score는 Data + Learner 둘 다 받음)
🟠 가장 많이 틀리는 곳: Linear Regression → Test and Score를 이으면 자동으로 Learner→Learner가 연결돼야 해요. 혹시 다른 채널로 붙으면 연결선을 더블클릭해서 Learner → Learner로 맞추세요.

4 단계별 따라하기 👣

1

데이터 표로 확인

Data Table · Data
DatasetsData ▸──▶▸ DataData Table
  1. Data Table을 놓고 연결 → 더블클릭.
  2. TV·Radio·Newspaper(광고비)와 맨 끝 Sales(판매량) 열을 확인.

Sales 옆에 ‘target’ 표시가 있으면, 이게 우리가 맞힐 정답이에요.

2

관계 눈으로 보기 (+회귀선)

Scatter Plot · Visualize
DatasetsData ▸──▶▸ DataScatter Plot
  1. Scatter Plot을 놓고 Datasets와 연결 → 더블클릭.
  2. Axis x: TV, Axis y: Sales 로 설정.
  3. 오른쪽 옵션에서 ‘Show regression line(회귀선 보기)’ 체크 → 점들 사이로 직선이 그어져요!

🌟 TV 광고비가 커질수록 판매량도 오르는 우상향 직선이 보이죠? 이게 회귀의 핵심 아이디어예요.

3

예측 모델 학습

Linear Regression · Model
DatasetsData ▸──▶▸ DataLinear Regression
  1. Linear Regression을 놓고 Datasets와 연결. (왼쪽 Model 칸)
  2. 광고비 → 판매량을 잇는 가장 잘 맞는 직선(공식)을 스스로 찾아요.
4

정확도 재기 (R²)

Test and Score · Evaluate
DatasetsData ▸──▶▸ DataTest and Score
Linear RegressionLearner ▸──▶▸ LearnerTest and Score
  1. Test and Score를 놓고 두 줄을 위처럼 연결. (왼쪽 Evaluate 칸)
  2. 더블클릭 → 왼쪽 Cross validation 선택 → 평가 점수가 표로 나와요.
R²
1에 가까울수록 잘 맞음 (예: 0.9 = 아주 좋음)
RMSE / MAE
예측이 평균 얼마나 빗나갔나 (작을수록 좋음)

5 도전 & 확인하기 ✅

  • 우리 모델의 R²는 얼마인가요? (1에 가까운가요?)
  • Scatter Plot에서 x를 Radio, Newspaper로 바꿔보기 — 어떤 광고가 판매량과 가장 관계가 셀까?
  • Linear Regression의 Coefficients 출력을 Data Table에 연결해, 어떤 광고의 영향이 큰지 숫자로 보기 (도전)
  • 회귀(숫자 예측)와 분류(종류 맞히기)의 차이를 한 문장으로 적어보기

6 실습 결론 질문 🧐 (실습 끝나고 답해보기)

Q1. 광고비가 늘어날수록 판매량은 어떻게 변했나요? (Scatter Plot 회귀선의 방향으로 설명)

Q2. TV·Radio·Newspaper 중 판매량과 가장 관계가 강한 광고와 가장 약한 광고는 무엇이었나요?

Q3. 우리 모델의 R²는 얼마였고, 그 숫자는 무엇을 뜻하나요?

Q4. 회귀가 분류와 다른 점은 무엇이며, 이 실습으로 알게 된 회귀의 핵심을 한 문장으로?

제출하면 선생님 구글시트로 바로 전송돼요. (한 번만 제출)

7 자주 나오는 문제 (교사용) 🛟

🟢 펼쳐서 보기 — 잘 안 될 때 점검
Datasets 목록이 안 떠요
인터넷 연결 확인(처음 받을 때 온라인 필요). 검색창에 advertising.
Test and Score에 점수가 안 나와요
Datasets→Data와 Linear Regression→Learner가 둘 다 연결됐는지 확인.
분류 점수(CA)만 보이고 R²가 없어요
정답(Sales)이 숫자형 target일 때만 회귀 점수가 나와요. advertising은 Sales가 target이라 정상이면 R²/RMSE가 보여요.
연결이 Learner가 아니라 다른 데로 붙어요
연결선을 더블클릭 → Learner → Learner로 맞추기.
회귀선이 안 보여요
Scatter Plot 옵션에서 Show regression line 체크, x·y가 숫자 변수인지 확인.
