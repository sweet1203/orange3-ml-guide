🌸 Orange3로 붓꽃 끼리끼리 묶기

품종을 안 알려줘도 AI가 닮은 꽃끼리 묶는 걸 보고, 실제 품종과 비교한다 — 비지도(군집) · k-평균

🍊 도구: Orange3 📦 데이터: iris (내장) 🧩 Datasets 위젯 ⏱️ 약 50분 ⌨️ 코딩 0줄
🏠 실습 목록으로
🎯 오늘의 목표 — 붓꽃의 꽃잎·꽃받침 크기만 보고, 품종(정답)을 안 알려줘도 AI가 비슷한 꽃끼리 묶는 걸 확인하고, 그 묶음이 실제 품종과 얼마나 맞는지 비교한다.

1 어떤 데이터를 쓰나요? 🌸

iris(붓꽃) — 머신러닝에서 가장 유명한 연습 데이터예요. 붓꽃 150송이, 각 꽃마다 크기 4가지가 적혀 있어요.

크기 특징 4가지 (입력)

꽃받침 길이 sepal length꽃받침 너비 sepal width🌟 꽃잎 길이 petal length🌟 꽃잎 너비 petal width

실제 품종 (있긴 하지만 오늘은 숨기고 시작!)

setosaversicolorvirginica
👉 오늘은 품종을 안 보고 크기만으로 묶어요(= 비지도학습·군집). 다 묶은 뒤에 “실제 품종과 맞나?”를 확인하는 게 핵심 재미!
🟢 File 위젯 말고 Datasets 위젯! · File=내 컴퓨터 파일 / Datasets=Orange 제공 예제(오늘 사용)

2 Datasets 위젯으로 불러오기 📥

  1. 왼쪽 Data 칸에서 Datasets 위젯을 캔버스에 놓기. (File 아님!)
  2. 더블클릭 → 예제 목록(인터넷 필요) → 검색창에 iris 입력.
  3. iris 줄을 더블클릭 → 자동 다운로드(구름→체크).
  4. 아래에 150 instances가 보이면 성공! 이제 Data로 내보낼 준비 완료.
🌐 처음 받을 때만 인터넷 필요(서버: datasets.biolab.si). iris는 자주 쓰여 캐시에 남아요.

4 단계별 따라하기 👣

1

데이터 표로 확인

Data Table · Data
DatasetsData ▸──▶▸ DataData Table
  1. Data Table을 놓고 연결 → 더블클릭.
  2. 꽃 150송이의 크기 4가지를 확인. (품종 열 iris도 보이지만, 군집엔 안 써요!)
2

묶기 전 점 보기

Scatter Plot · Visualize
DatasetsData ▸──▶▸ DataScatter Plot
  1. Scatter Plot을 놓고 Datasets와 연결 → 더블클릭.
  2. Axis x: petal length, Axis y: petal width (꽃잎 길이·너비가 품종을 가장 잘 가름!).
  3. 점들이 흩어져 보여요. (Color는 아직 끄거나 같은 색으로 두기)
3

군집 만들기

k-Means · Unsupervised
DatasetsData ▸──▶▸ Datak-Means
  1. k-Means를 놓고 Datasets와 연결. (왼쪽 Unsupervised 칸)
  2. 더블클릭 → Number of Clusters: 3 (Fixed에 3). Re-runs는 그대로 두면 여러 번 시도해 더 좋은 군집을 찾아줘요.
  3. 데이터에 ‘Cluster’ 정보가 붙어 출력돼요.
4

묶은 결과 색으로 보기

Scatter Plot (1) · Visualize
k-MeansData ▸──▶▸ DataScatter Plot (1)⬅ 이번엔 k-Means에서 연결!
  1. 새 Scatter Plot을 놓고 k-Means와 연결.
  2. 더블클릭 → x: petal length, y: petal width, Color: Cluster.
  3. 🎉 꽃들이 세 가지 색으로 묶였어요!
5

🌟 정답과 맞춰보기 (오늘의 하이라이트!)

같은 Scatter Plot (1)
  1. 방금 그 Scatter Plot (1)에서 Color를 Cluster → iris(실제 품종)로 바꿔보세요.
  2. AI가 묶은 색(Cluster)과 실제 품종 색이 거의 똑같이 겹치죠?

🌟 정답을 한 번도 안 알려줬는데, AI가 묶은 무리가 실제 붓꽃 품종과 거의 일치해요! 이게 비지도학습의 놀라운 점이에요.
